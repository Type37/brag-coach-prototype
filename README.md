# Brag Coach Prototype

Interactive prototype for the Destination Cleveland "Brag Coach" AI assistant concept.

## What this is

A clickable prototype showing how an AI-powered brag/hype tool could integrate into [thisiscleveland.com](https://www.thisiscleveland.com/). The assistant generates captions, comebacks, DMs, and TikTok concepts with Cleveland's voice.

## Features

- **Desktop:** Floating search pill + site nav icons with merge animation on open
- **Mobile:** Bottom sheet with tabbed SEARCH / BRAG COACH modes
- **Context-aware:** Prompts change per page (Homepage, Events, Downtown, Mabel's BBQ, Music)
- **Multiple output formats:** Captions, comebacks (per-item copy), DM templates, TikTok concepts
- **Deflection handling:** Off-topic queries get redirected with personality

## Usage

Static HTML file. No build step. Open `index.html` or deploy to GitHub Pages.

Use the prototype switcher bar at top to change pages and toggle Desktop/Mobile views.
